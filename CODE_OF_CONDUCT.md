# 行为准则

## 我们的承诺

为了营造一个开放、友好的环境，我们作为贡献者和维护者承诺：无论年龄、体型、身体健全与否、民族、性征、性别认同与表达、经验水平、教育程度、社会地位、国籍、相貌、种族、宗教信仰、性取向，我们均以友善和尊重的态度对待项目社区中的每一位成员。

## 我们的标准

有助于创造积极环境的行为包括但不限于：

* 使用友善和包容的语言
* 尊重不同的观点和经历
* 耐心地接受建设性批评
* 关注对社区最有利的事情
* 友善对待其他社区成员

不当行为包括但不限于：

* 使用与性有关的言语或是图像，以及不受欢迎的性关注
* 捣乱/煽动/造谣的行为，侮辱/贬损的评论，及人身攻击或是政治攻击
* 公开或私下的骚扰
* 未经许可地公布他人的资料，如住址、电子邮箱等
* 其他有理由认定为违反职业操守的不当行为

## 我们的责任

项目维护者有责任为「可接受的行为」标准做出诠释，以及对已发生的不当行为采取恰当且公平的纠正措施。

项目维护者有权利和义务去删除、编辑、拒绝违背本行为标准的评论（comments）、提交（commits）、代码、wiki 编辑、问题（issues）等贡献；如有必要，可暂时或永久性地封禁任何他们认为行为不当、威胁、冒犯、有害的参与者。

## 适用范围

本行为标准适用于本项目中的所有场合，以及在本项目中代表社区时的公开场合。

代表社区的情况包括但不限于：使用官方项目邮箱、通过官方社交媒体账号发言、作为指定代表参与在线或线下活动等。

## 如何报告问题

如果您遭遇或发现任何违反本行为准则的行为，请通过以下方式报告：

1. **GitHub Issues**：创建一个新的 Issue，选择「问题咨询」模板
2. **直接联系**：发送邮件至项目维护者

所有投诉都将得到及时和公正的审查和调查。项目维护者有义务对事件报告者保密。

## 处理流程

1. **初步审查**：维护者将在24小时内确认收到报告
2. **调查**：维护者将进行调查，可能需要与相关方沟通
3. **决定**：基于调查结果，维护者将做出决定
4. **执行**：采取适当的纠正措施
5. **反馈**：向报告者反馈处理结果（在保护隐私的前提下）

## 纠正措施

根据违规行为的严重程度，可能采取的措施包括：

1. **警告**：书面警告，说明违规行为
2. **暂时封禁**：暂时禁止参与社区活动
3. **永久封禁**：永久禁止参与社区活动
4. **内容移除**：删除违规内容
